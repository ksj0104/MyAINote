# TERMINAL//BREACH — 스토리 설계 문서

> 이 문서는 시나리오(메인) 모드의 서사 백본이다. 대사 작성·미션 추가 시 여기에 맞춘다.
> 인게임 구현: 각 레벨의 `interlude`(막간극) · `brief`(브리핑) · `incoming`(접선) · `success`(디브리핑),
> 그리고 `intel` 명령(작전 브리핑 화면)으로 전달된다.

---

## 1. 로그라인 / 테마

> **거대기업이 도시의 빛과 정보를 모두 쥔 시대, 이름 없는 신입 해커가 멘토의 지도 아래
> 기초 터미널에서 시작해 메가코프 HELIOS의 심장을 꿰뚫고 — 도시를 사슬에서 풀어낸다.**

- **중심 테마**: 정보는 가두는 순간 썩는다. 우리는 훔치지 않는다 — 잠긴 진실을 풀어줄 뿐.
- **개인 테마**: 두려움은 약점이 아니라 생존 본능이다. 신입 → 요원 → 전설로의 성장.
- **도덕적 긴장(ACT II)**: 도시 전력망을 쥔다는 것 — 무기로 쓸 것인가, 증거로 쓸 것인가.

---

## 2. 세계관

근미래. 메가코프 **HELIOS** 가 도시의 전력·데이터·감시를 독점한다. 모든 선은 그들의 코어로 모인다.
**NULLSEC** 은 감시받지 않는 마지막 사람들 — 이름 없는 해커 셀. 거대기업의 틈새에 산다.

---

## 3. 등장 인물 / 세력

| | 정체 | 역할 | 아크 |
|---|---|---|---|
| **플레이어** | NULLSEC 신입 | 주인공(이름 없음) | 신입(L1) → 요원(L13~) → 전설(엔딩) |
| **0xMOTHER** | NULLSEC 베테랑 핸들러 | 멘토·화자 | 냉소적 보호자. 끝에 "넌 더 이상 내 학생이 아니야" |
| **HELIOS** | 메가코프 | 표적 | 코어 함락(ACT I) → 그림자로 후퇴(ACT II) |
| **ORACLE** | HELIOS 방어 AI(사냥개) | 적대자 | 도입(L10) → 고조(L13) → 압박(ACT II) → 침묵(엔딩) |
| **WRAITH** | 정체불명의 해커(다크아미풍) | 트릭스터 | 접선(L14) → 유혹 → 속편 떡밥(엔딩) |

**ORACLE 회수**: 침입 패턴을 학습·되갚는 자동 방어. 후반부 "속도·흔적제거" 긴장의 근원.
엔딩에서 게이트의 불이 꺼지며 **신호가 끊긴다** = 플레이어가 ORACLE을 끝내 앞질렀다는 증거.
**WRAITH 회수**: "0xMOTHER는 절반만 말한다." 믿지 말되 무시도 말라. 엔딩의 마지막 신호로 다음 판을 예고.

## 4. 3막 구조 (20 미션)

### ACT I — 코어를 향하여 (L01–L13)
신입의 훈련과 첫 전쟁. 기초기 → 첫 네트워크 → 권한상승 → HELIOS 코어 함락으로 **전설 등극**.

- **기초 (L01–L05)** — 입단·훈련. 0xMOTHER가 그릇을 시험. ls/cat → ls -a → grep → chmod → base64.
- **중급 (L06–L09)** — 첫 실전 네트워크. nmap 정찰 → ssh 침투 → nmap -sV → auth.log 분석.
- **고급 (L10–L13)** — **ORACLE 등장**. sudo 권한상승 → john 해시크랙 → 피벗 → **HELIOS 코어 함락**. (L13: 신입→요원)

### ACT II — 그림자 전쟁 (L14–L20)
HELIOS는 죽지 않았다. 더 깊은 어둠으로 숨었을 뿐. **WRAITH 접선**, 도덕적 무게, 그리고 블랙아웃.

- **실전 (L14–L20)** — hydra 브루트 → xor 복호 → DB 추출 → 코드 탈취 → 백도어 → **무선 거점** → **오퍼레이션 블랙아웃(피날레)**.

## 5. 미션별 비트 (서사 목적 ↔ 학습 목표)

| # | 미션 | 서사 비트 | 배우는 것 |
|---|---|---|---|
| L01 | 첫 접속 | 입단. "겁먹고 달아나면 존재한 적 없는 사람이 된다" | ls, cat, submit |
| L02 | 숨겨진 흔적 | 보이지 않는 걸 보는 첫 본능 | ls -a |
| L03 | 바늘 찾기 | 양(量)이 무기인 싸움 — 노가다 vs 패턴 | grep -r |
| L04 | 잠긴 문 | "주인은 제 자물쇠를 연다" | chmod |
| L05 | 암호문 | "인코딩은 커튼일 뿐" — 기초 졸업 | base64 -d |
| L06 | 정찰 | 골방을 나서다. "정찰 없는 침투는 자살" | nmap /24 |
| L07 | 첫 침투 | 첫 남의 집 — 첫 침투의 맛 | cat creds → ssh |
| L08 | 서비스 식별 | 버전이 곧 입장권 | nmap -sV |
| L09 | 침입자 추적 | 방어도 공격의 일부 | grep auth.log |
| L10 | 권한 상승 | **ORACLE 경고** — 규칙이 바뀐다 | sudo cat shadow |
| L11 | 해시 크래킹 | "신의 자리는 짧고 위험하다" | john → su root |
| L12 | 피벗 | 발판을 다리 삼아 심장부로 | jumpbox → DB |
| L13 | **최종: HELIOS** | **코어 함락 · 신입→요원 · 전설 등극** | 풀체인 |
| L14 | 워드리스트 브루트 | **ACT II 개전 · WRAITH 접선** | hydra → ssh |
| L15 | 암호문 복호화 | 적의 속삭임을 읽다 | xor |
| L16 | 원격 DB 추출 | 작은 틈이 왕국을 무너뜨린다 | dump → hashcat |
| L17 | 원격 코드 탈취 | 적의 무기로 적을 친다(ORACLE 우회) | grep -r API_KEY |
| L18 | 백도어 삽입 | 끊겨도 돌아오는 발판 | echo >> rc.local |
| L19 | **무선 거점** | 그리드에 물리적으로 붙다 | airmon→airodump→aireplay→aircrack |
| L20 | **오퍼레이션 블랙아웃** | **피날레 · 도덕적 선택 · 그리드 장악** | 풀체인 보스 |

## 6. 몰입 장치 (인게임)

- **막간극(interlude)** — 티어 전환점의 컷신: L01 입단 · L06 첫 네트워크 · L10 ORACLE · L13 코어 · L14 ACT II · L20 피날레.
- **brief / success** — 매 미션 0xMOTHER의 브리핑(4~6줄)과 디브리핑(2~3줄). 앞 단계를 언급해 연속감.
- **incoming** — WRAITH의 강제 접선(L14). 연락처에 추가됨.
- **chat / channel** — 0xMOTHER·WRAITH와 실시간 대화(키워드 응답 + 맥락형 도움).
- **intel** — 작전 브리핑 화면: 작전 개요 · 현재 단계(ACT) · **현재 목표** · 세력 현황(NULLSEC/HELIOS/ORACLE 경계도/WRAITH) · 행적. *길을 잃지 않고 목표를 향하게 하는 길잡이.*

---

## 7. 톤 & 스타일 가이드

- 0xMOTHER: 짧고 단단한 문장. 냉소 + 다크 유머. 겉은 무뚝뚝, 속은 보호. 가르칠 땐 비유.
- WRAITH: 소문자·말줄임·불신. "검증해 — 늘." 절대 다 말하지 않는다.
- 호칭: L01–L12 "신입" → L13 이후 "요원".
- 모든 해킹은 **시뮬레이션**. 윤리는 해커의 무기다 — 엔딩이 못박는 메시지.
