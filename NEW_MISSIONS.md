## 0. 로그라인 & 테마

> **거대기업 HELIOS가 도시의 빛과 정보를 모두 쥔 시대 — 정체 모를 멘토에게 터미널을 빌려준 신입 해커가,
> 기초 명령에서 코어 장악까지 올라가며 '누구를 믿을 것인가'를 배운다. 도시를 끄는 마지막 키 앞에서,
> 자신이 어떤 해커가 될지 스스로 정한다.**

- **중심 테마**: 정보는 가두면 썩는다. 우리는 훔치지 않는다 — 잠긴 진실을 풀어줄 뿐.
- **개인 테마**: 신뢰는 검증의 대상이다. 멘토도, 유혹도, 권력도.
- **도덕 축(ACT 후반)**: 같은 root, 다른 길 — 신념(0xMOTHER) ↔ 효율(WRAITH). 선택이 누적되어 결말이 갈린다.

## 2. ⚖ 성향(STANCE) 시스템

WRAITH가 던지는 **지름길**을 택할 때마다 `GHOST(유령)` 성향이, 0xMOTHER의 **정도**를 따를 때마다 `CREED(신념)` 성향이 쌓인다.
누적치가 **엔딩 분기**를 결정한다. (인게임: `intel` 화면에 STANCE 게이지 + ORACLE 경계도 표시)

| 선택 예 | 결과 |
|---|---|
| S3.02 크론잡 백도어(WRAITH) vs SUID 정공법(MOTHER) | GHOST+ / CREED+ |
| S3.08 블랙리스트 **파괴**(WRAITH) vs **복사 후 보존**(MOTHER) | GHOST++ / CREED++ |
| S4 흔적 처리 강도, 코어 종료 방식 | 최종 분기 가중 |

> 지름길은 빠르지만 **ORACLE 경계도**를 덜 올리는 대신 **윤리·신뢰**를 깎는다. 정공법은 느리지만 깨끗하다.

# 🎬 Series 1 — 각성 (The Awakening)
**테마**: 기초 명령 체득 + OSINT(공개정보 정찰) · **ORACLE: DORMANT**

> **콜드 오픈** — 새벽 3시. 잠긴 화면이 저 혼자 깨어난다.
> `[UNKNOWN]` *"네 터미널의 권한을 잠시 빌렸다. 비명 지르지 마. 지금부터 30초, 내 말만 들으면 넌 안 잡혀."*
> **신규 도구**: `ls cat ls -a grep chmod base64 dig nmap curl nmap -sV hydra`

#### S1.01 · 눈을 떠라 〔EYES OPEN〕
> 빌려준 터미널. 정체불명의 손이 첫 지시를 내린다.
- **📩 UNKNOWN** — "환영할 시간은 없다. 그들이 널 보고 있어. 디렉터리에 남긴 `readme.txt`부터 읽어. 천천히 — 떨어도 좋으니 멈추지만 마라."
- **🎯 목표**: `readme.txt`를 읽어 첫 지시를 확인한다.
- **🧠 스킬**: `ls`(둘러보기) · `cat`(읽기)
- **💡 힌트**: ① `ls`로 파일을 본다 ② `cat readme.txt`
- **⌨ 풀이**: `ls` → `cat readme.txt`
- **🎁 전리품**: 첫 접속 코드 · 화자가 `0xMOTHER`로 자기를 밝힌다.
- **🛡 방어**: 최소권한·세션 격리 — 빌린 셸이 가능한 건 권한 위임이 허술해서다.

#### S1.02 · 숨겨진 흔적 〔HIDDEN TRACES〕
> HELIOS는 모든 걸 기록한다. 보이는 게 다가 아니다.
- **📩 0xMOTHER** — "겉보기에 속지 마. 점(.)으로 시작하는 파일은 눈을 피해 숨는다. 모든 걸 보는 법을 배워."
- **🎯 목표**: 숨김 설정파일 `.env`를 찾아 그 안의 자격증명을 확보.
- **🧠 스킬**: `ls -a`(숨김 표시)
- **💡 힌트**: ① `ls -a` ② 점으로 시작하는 이름을 `cat`
- **⌨ 풀이**: `ls -a` → `cat .env`
- **🎁 전리품**: 평문 API 키/DB 자격증명 한 줄.
- **🛡 방어**: 비밀은 코드/`.env`에 두지 말 것 — 시크릿 매니저·환경 분리.

#### S1.03 · 바늘 찾기 〔NEEDLE〕
> 수십만 줄의 더미 로그. 손으로는 평생.
- **📩 0xMOTHER** — "패턴은 거짓을 모른다. 이 산더미에서 `HELIOS`가 박힌 줄만 끌어내. grep은 해커의 자석이다."
- **🎯 목표**: `system.log`에서 `HELIOS` 포함 줄을 찾아 핵심 정보를 제출.
- **🧠 스킬**: `grep`(패턴 검색)
- **💡 힌트**: ① `grep "HELIOS" system.log` ② 여러 줄이면 자격증명 형태를 골라라
- **⌨ 풀이**: `grep "HELIOS" system.log`
- **🎁 전리품**: 내부 호스트명/계정 단서.
- **🛡 방어**: 로그에 민감정보 적재 금지 · 로그 마스킹.

#### S1.04 · 권한의 무게 〔THE WEIGHT OF rwx〕
- **📩 0xMOTHER** — "어떤 바보가 관리 스크립트의 실행권한을 막아뒀군. 자물쇠는 권한일 뿐 — 주인은 언제든 연다."
- **🎯 목표**: `start.sh`에 실행권한을 주고 실행.
- **🧠 스킬**: `chmod`(권한) · `./` 실행
- **💡 힌트**: ① `ls -l`로 권한 확인(`-rw-`) ② `chmod +x start.sh` ③ `./start.sh`
- **⌨ 풀이**: `chmod +x start.sh` → `./start.sh`
- **🎁 전리품**: 스크립트가 내부 정찰 좌표를 출력.
- **🛡 방어**: 실행권한·소유권 점검, 신뢰되지 않은 스크립트 실행 금지.

#### S1.05 · 변환된 암호 〔THE CURTAIN〕
- **📩 0xMOTHER** — "암호화가 아니야 — 그냥 알아보기 힘들게 입힌 옷이지. base64는 가장 흔한 위장. `==`로 끝나면 십중팔구 그놈이다."
- **🎯 목표**: 발견한 base64 문자열을 디코딩.
- **🧠 스킬**: `base64 -d`(인코딩 ≠ 암호)
- **💡 힌트**: ① `cat`으로 이상한 문자열 확인 ② `echo "<문자열>" | base64 -d`
- **⌨ 풀이**: `echo "SGVsaW9zQWRtaW4=" | base64 -d` → `HeliosAdmin`
- **🎁 전리품**: 관리자 사용자명 `HeliosAdmin`.
- **🛡 방어**: 인코딩을 보안으로 착각 말 것 — 실제 암호화·접근통제 필요.

#### S1.06 · 그림자 도메인 〔SHADOW DOMAIN〕
- **📩 0xMOTHER** — "거인은 발톱의 때를 신경 안 써. 네임서버를 찔러 외부로 새어나온 테스트 서브도메인을 캐내라."
- **🎯 목표**: DNS에서 노출된 서브도메인 `dev-test.helios.corp` 발견.
- **🧠 스킬**: `dig`(DNS 정찰 · 존 트랜스퍼)
- **💡 힌트**: ① `dig helios.corp` ② 잘못 열린 AXFR를 노린다
- **⌨ 풀이**: `dig axfr helios.corp @ns.helios.corp`
- **🎁 전리품**: 테스트 서브도메인 + 내부 IP 힌트.
- **🛡 방어**: AXFR 제한 · 내부 도메인 분리 · DNS 최소 노출.

#### S1.07 · 네트워크 스캔 〔KNOCK KNOCK〕
- **📩 0xMOTHER** — "문이 열렸는지 두드려 봐야지. 스캐너를 돌려 살아있는 포트를 그려라. 우측 네트워크 맵을 봐."
- **🎯 목표**: `dev-test.helios.corp`의 열린 포트 확인.
- **🧠 스킬**: `nmap`(포트 스캔)
- **💡 힌트**: ① `nmap dev-test.helios.corp` ② 21/22/80을 주목
- **⌨ 풀이**: `nmap dev-test.helios.corp`
- **🎁 전리품**: 21(FTP)·22(SSH)·80(HTTP) 오픈.
- **🛡 방어**: 불필요 포트 차단 · 외부 노출 최소화 · 포트노킹.

#### S1.08 · 이메일 스크래핑 〔FIRST WEAPON〕
- **📩 0xMOTHER** — "팀 소개 페이지 소스를 긁어 직원 이메일만 추려내. 이게 우리의 첫 무기다."

- **🧠 스킬**: `curl`(웹 가져오기) · `grep -oE`(정규식 추출) · `>`(리다이렉션)

- **⌨ 풀이**: `curl dev-test.helios.corp/team | grep -oE "[a-zA-Z0-9._]+@helios.corp" > emails.txt`
- **🎁 전리품**: `emails.txt`(이후 시리즈에서 재사용 — **연속성**).
- **🛡 방어**: 직원정보 공개 최소화 · OSINT 표면 관리.

#### S1.09 · 버전 식별 〔OLD SHIELDS〕
- **📩 0xMOTHER** — "무작정 들이받지 마. 버전을 알아내. 낡은 방패는 쉽게 깨진다 — 취약점은 버전에 붙으니까."

- **🎁 전리품**: 구버전 FTP/SSH 식별 → CVE 단서.
- **🛡 방어**: 패치·버전 은닉(배너 그래빙 방지)·EOL 서비스 제거.

#### S1.10 · 첫 번째 침투 〔FIRST BLOOD〕
- **📩 0xMOTHER** — "모은 이메일을 봐. 대부분 초기 비번을 안 바꿨어. 낡은 FTP 문을 부숴. 넌 이제 구경꾼이 아니다."

# 🎬 Series 2 — 침투 (The Infiltration)
**테마**: 웹 취약점 · 내부망 거점 · **ORACLE 각성** · **WRAITH 첫 접선**

> **콜드 오픈** — 탈취 문서에서 내부 헬프데스크 망의 존재가 드러난다. 대문은 잠겼지만, 모든 자물쇠엔 틈이 있다.
> **신규 도구**: `robots.txt` · SQLi(`' OR '1'='1`) · UNION · LFI · `tcpdump` · 파일 업로드 웹쉘 · `nc` 리버스셸

#### S2.01 · 로봇의 규칙 〔robots.txt〕
- **📩 0xMOTHER** — "검색 로봇에게만 알려주는 '들어오지 마' 팻말이 있어. 거기 숨길 게 적혀 있지."
- **🎯 목표**: `robots.txt`에서 숨겨진 경로(관리자 페이지) 발견.
- **🧠 스킬**: `curl` · 정보노출 분석
- **💡 힌트**: ① `curl http://help.helios.corp/robots.txt` ② `Disallow:` 경로를 노려라
- **⌨ 풀이**: `curl http://help.helios.corp/robots.txt`
- **🎁 전리품**: 숨겨진 로그인 경로 `/admin_portal`.
- **🛡 방어**: robots.txt에 비밀경로 적지 말 것 — 차단은 인증으로.

#### S2.02 · 거짓말쟁이 〔THE LIAR〕
- **📩 0xMOTHER** — "DB는 멍청해. 네 말을 무조건 '참'이라 믿게 만들어. 인증을 *논리로* 뚫는 거다."
- **🎯 목표**: SQL 인젝션으로 인증 우회 로그인.
- **🧠 스킬**: **SQLi 기초**(`' OR '1'='1`)
- **💡 힌트**: ① 입력칸이 쿼리에 그대로 들어간다 ② 항상 참이 되는 조건을 주입 ③ 주석(`--`)으로 뒷부분 무력화
- **⌨ 풀이**: `login admin "' OR '1'='1' --"`
- **🎁 전리품**: 저권한 관리 세션.
- **🛡 방어**: **Prepared Statement/파라미터 바인딩** · 입력검증 · WAF.

#### S2.03 · 깊은 곳의 장부 〔UNION〕
- **📩 0xMOTHER** — "권한이 낮군. 인젝션을 *더 길게* 이어서, 사용자 테이블의 해시를 화면에 토하게 만들어."
- **🎯 목표**: UNION 기반 데이터 추출로 `users` 해시 덤프.
- **🧠 스킬**: **UNION-based SQLi**
- **💡 힌트**: ① 컬럼 수를 맞춰라 ② `UNION SELECT`로 다른 테이블을 붙인다
- **⌨ 풀이**: `' UNION SELECT null, username, password FROM users --`

- **🎯 목표**: SSH 포트포워딩으로 분리망 코어 DB 접근.
- **🧠 스킬**: `ssh -L`(로컬 포워딩) · 피벗
- **💡 힌트**: ① `ssh -L 8080:10.0.0.1:80 admin@10.0.1.1` ② `localhost:8080`으로 접근
- **⌨ 풀이**: `ssh -L 8080:10.0.0.1:80 admin@10.0.1.1` → `curl localhost:8080`
- **🎁 전리품**: 코어 DB 도달.
- **🛡 방어**: 망분리·점프호스트 통제 · 포워딩 비활성 · 마이크로세분화.

#### S4.06 · 디지털 세탁 〔LAUNDRY〕 ⚖
- **📩 0xMOTHER** — "셧다운 전 마지막 경고다. 수십만 줄 로그에서 네 IP(10.0.0.42)가 찍힌 줄만 *단 한 줄도 남김없이* 도려내라. 안 그러면 평생 쫓긴다."
- **🎯 목표**: 자기 흔적(IP)을 로그에서 정밀 제거.
- **🧠 스킬**: `sed -i`(인플레이스 편집) · 안티 포렌식
- **💡 힌트**: ① `grep`으로 자기 IP 줄 확인 ② `sed -i '/10.0.0.42/d' /var/log/syslog`
- **⌨ 풀이**: `sed -i '/10.0.0.42/d' /var/log/syslog`
- **🎁 전리품**: 흔적 1차 소거.(처리 강도에 따라 STANCE 가중)
- **🛡 방어**: **불변/원격 로그**(WORM·SIEM 전송) · 로그 무결성 서명.

#### S4.07 · 시간 조작 〔TIMESTOMP〕
- **📩 WRAITH** — "로그만 지운다고 끝이 아니야. 파일 수정 시간을 1년 전으로 돌려. 포렌식은 *시간*을 본다."
- **🎯 목표**: 타임스탬프 변조로 포렌식 교란.
- **🧠 스킬**: `touch -t`(타임스톰핑)
- **💡 힌트**: ① `stat`으로 시간 확인 ② `touch -t 202501010000 /var/log/syslog`
- **⌨ 풀이**: `touch -t 202501010000 /var/log/syslog`
- **🎁 전리품**: 타임라인 교란 완료.
- **🛡 방어**: 변경 불가 타임스탬프(원격 기록) · 파일 무결성 모니터링(inotify).

#### S4.08 · 블랙아웃 〔TOTAL BLACKOUT〕 ⚖⚑
- **📩 ORACLE** — "경고. 당신의 행동은 도시 인프라의 **74%**를 마비시킵니다. 시스템을 종료하시겠습니까?"
- **📩 0xMOTHER** — "플러그를 뽑아."
- **🎯 목표**: 코어를 강제 종료 — **도시 정전**. 그러나 *어떻게* 끝낼지는 그동안의 STANCE가 정한다.
- **🧠 스킬**: `systemctl`(전원 제어) · 결단
- **💡 힌트**: ① 마스터 차단기 = 코어 전원 ② `systemctl poweroff -f`
- **⌨ 풀이**: `systemctl poweroff -f`
- **🎁 전리품**: **엔딩 분기 트리거** ↓
- **🛡 방어**: 중요 인프라 격리 · 다중 승인(2-person) · 물리 페일세이프.

# 🎯 엔딩 (STANCE 분기)

마지막 명령은 같아도, 누적된 성향이 *결말의 의미*를 가른다.

### ⓐ 충직한 칼날 — `CREED` 우세
정도를 지키고 블랙리스트를 *증거로* 보존했다. 도시의 불이 꺼지고 — 동시에 전 세계 뉴스에 HELIOS의 범죄가 송출된다.
`0xMOTHER`: "넌 날 믿었지. …난 잡혀가겠지만, 진실은 남는다. 잘 가라, 요원." → 그녀는 사라지지만 **전설**이 된다.

### ⓑ 유령 — `GHOST` 우세
WRAITH의 지름길을 거듭 택했다. 흔적 없이 깨끗이. 0xMOTHER의 명단은 재가 됐고, 그녀의 행방은 묘연하다.
`WRAITH`: "넌 이제 유령이야. 이름도, 양심의 무게도 없는. …환영한다." → 플레이어는 **새로운 사냥꾼**이 된다.

### ⓒ 제3의 길 — 균형
어느 쪽도 완전히 따르지 않았다. 너는 도시를 끄지 않고 — *증거만 흘리고 코어를 살린다.*
`0xMOTHER`: "…네가 우릴 능가했군." `WRAITH`: "흥미롭네. 다음엔 내 편이 되어줘." → **열린 결말**, 속편 떡밥.

# 부록 A — 엔진 구현 매핑 (현 게임 기준)

> 현재 게임 엔진(`js/commands.js`)에 **이미 있는** 명령: `ls cat grep find chmod base64 nmap nmap -sV ssh hydra john hashcat dump xor caesar vigenere rot13 strings md5sum sudo su scp wget curl echo(>,>>) airmon-ng airodump-ng aireplay-ng aircrack-ng` 등.
>
> 이 설계서를 구현하려면 **신규 시뮬 명령/메커니즘**이 필요하다:

| 신규 요소 | 구현 메모 |
|---|---|
| `dig`(AXFR) | DNS 레코드/서브도메인을 레벨 데이터로, 존 트랜스퍼 시뮬 |
| SQLi(`' OR '1'='1'`, UNION) | 가상 로그인/쿼리 핸들러 — 입력 패턴 매칭으로 우회/덤프 |
| LFI(`?file=../`) | curl 핸들러에 경로조작 → 가상 FS 파일 반환 |
| `tcpdump -r` | 미리 만든 pcap 텍스트에서 grep |
| 파일 업로드 + 웹쉘 + `nc -lvnp` | 업로드→웹쉘 경로→리스너 콜백 상태머신 |
| `find -perm -4000` | SUID 플래그를 노드 메타에 추가 |
| cron 조작(`>> /opt/health.sh`) | 쓰기 후 "1분 후 root" 트리거 |
| 다중 압축(`bunzip2/tar/gunzip`) | 파일에 `archive` 메타 체인 |
| `steghide extract` | 이미지 노드에 숨김 페이로드 메타 |
| 도커 탈출(`mount /dev/sda1`) | 컨테이너 FS ↔ 호스트 FS 전환 |
| `ssh -L` 포워딩 | 피벗 + 가상 포트 맵 |
| `sed -i` / `touch -t` | 로그 노드 줄 삭제 / mtime 메타 변경 → 클리어 판정 |
| `systemctl poweroff` | 엔딩 트리거 |
| **STANCE 시스템** | `game.stance = {creed, ghost}`; 선택 미션에서 누적; `intel`에 게이지; 엔딩 분기 |
| **분기 미션** | 레벨에 `branches: [{label, cmd, effect}]` + 다중 `success`/다음 레벨 |
